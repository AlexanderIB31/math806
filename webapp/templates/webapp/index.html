Представляем вам сайт, на котором вы сможете найти численные решения уравнений математической физики: параболического, гиперболического и эллиптического типа. Каждое из уравнений имеет несколько методов решения, с различным порядком аппроксимации. Для большей наглядности и демонстрации качества работы каждого из алгоритмов, мы будем показывать вам графики функции решения каждого из методов, благодаря которым вы сам сможете оценить, какой из методов точнее всех решает данное уравнение. Также, вы сможете самостоятельно задавать необходимые параметры, например, мелкость разбиения сетки.
